❔ How do I connect to my real Pi? (step by step)
🔧 First time on this Pi — do these once:
  1. Copy the oivio-bridge folder onto your Raspberry Pi (ask your teacher where to save it — your home folder is fine).
  2. Open a Terminal (the black icon in the top bar, or press Ctrl+Alt+T).
  3. Install the helper library — type this and press Enter:
    sudo apt install -y python3-gpiozero python3-lgpio
  4. For the green LED only: turn SPI off so pin GPIO 7 is free. Type sudo raspi-config → Interface Options → SPI → No → Finish → reboot. (Red & yellow work without this.)
▶ Every time you want live hardware:
  1. Open a Terminal. If you saved the folder in your home folder, type cd oivio-bridge and press Enter. (Easier: in the File Manager, open the oivio-bridge folder, then choose Tools → Open Current Folder in Terminal — no typing needed.)
  2. Start the bridge:
    python3 autobotic_bridge.py
    (or, from anywhere without changing folder: python3 ~/oivio-bridge/autobotic_bridge.py). Leave this window open — you'll see "Listening on http://127.0.0.1:8076".
  3. Come back here and click 🔌 Connect my Pi below. Done!
⏏ When you're finished (or to run your own blink_led.py):
  1. Click ⏏ Release hardware below, or press Ctrl+C in the bridge window — that frees the pins for your own program.
Stuck? "Couldn't reach the bridge" → the bridge window isn't running (redo steps 5–6). Green LED won't switch off → SPI is still on (redo step 4).
